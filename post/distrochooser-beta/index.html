帮你选择 Linux 发行版的 Distrochooser，现在更加懂你

2020-04-02 4 min read # Linux

Distrochooser 是由德国人 Christoph Müller 推出的，帮助 Linux 用户选择适合自己的 Linux 发行版的网站。而现在，更智慧、更懂你的 Distrochooser 正在进行测试。

你可以访问 beta.distrochooser.de，体验新版 Distrochooser。（新版 Distrochooser 暂未提供中文版本）

但在此之前，让我们先来看看：

Distrochooser 如何了解我的需求？

Distrochooser 通过简单的 16 个问题，来了解你的 Linux 使用需求。你可以按顺序填写它们，也可以完全不按顺序。完成问卷后，你还可以更改各个问题的权重。同时，所有问题都不是必答题，如果相关选项完全不符合自己，你可以跳过问题。

完成问卷后，Distrochooser 将罗列适合你的发行版。接下来，自己选择吧！Distrochooser 就帮你到这里了。

当然，如果怕隐私问题，你也可以一键删除这些回答。

那么新的测试版多了什么？

更简便、专注的问卷

新的 Distrochooser 将问卷界面从列表式改为了一页一题式，使回答问卷更专注。同时，问题数量从 16 个 降到 13 个，整合了一些冗余的问题，使流程更快速。

冲突项提示

我相信有许多人会在填问卷时钻牛角尖的，比如前面选自己会一点点 Linux 知识，后面又吹嘘说自己绝大部分问题都能解决。现在，Distrochooser 会识别你的这些矛盾行为，并且给予你提示，算是对一些小学生的抨击了。

不过目前这个机制如果这么解释，也有一点点问题，比如“预装所有基本软件”和“拥有 Live Mode”被判为冲突。

更精细客观的推荐原因

以前，Distrochooser 仅罗列所有适合你的地方；现在，它更加客观了。在测试版里，Distrochooser 会详尽地列出几乎全部主流发行版的优点、缺点，同时也会显示不会推荐某个发行版的理由。虽然借助更客观的推荐原因，你可以更快地找到更适合自己的发行版，更快入坑 Linux。

如果这个发行版很符合你的需求，就给它一个小心心吧；如果不喜欢，那就点击“心碎”。当然，如果你觉得某个板块爱眼，也可以隐藏发行版的推荐原因。

为你的单个刚需“加星”

在原版中，你需要改变整个问题的权重来突出你的刚需；而在新版，你只需要在你的刚需旁点击星星，就可以提高单个需求的权重。同时在结果里，你的刚需也会出现星星，帮助你更快选择。
